# @hooks2022/use-title

React Hook to update your document's title.

## Installation

#### yarn

`yarn add @hooks2022/use-title`

#### npm

`npm i @hooks2022/use-title`

## Usage

```js
import React from "react";
import useTitle from "@hooks2022/use-title";

function App() {
  useTitle("Welcome");
  return <h1>Welcome</h1>;
}
```
